# .aerofortress — rede de conhecimento

Conhecimento curado deste repo, acumulado pelo [AeroFortress](https://github.com/lucas/aerofortress) (uma rede ME de contexto para agentes, inspirada no Applied Energistics 2).

- `items/<slug>/*.md` — os items (a verdade, versionada junto com o código)
- `index.db` — índice local regenerável (gitignored)

Clonou o repo e usa AeroFortress? Registre esta pasta como workspace e o conhecimento já estará na sua rede. ✦
